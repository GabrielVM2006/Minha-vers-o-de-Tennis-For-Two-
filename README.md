# Minha-versao-de-Tennis-For-Two-
Este projeto é uma recreação simples e minimalista do clássico jogo Tennis for Two, desenvolvido internamente em HTML, CSS e JavaScript puros. O objetivo é demonstrar lógica de colisão, movimento de objetos e manipulação de elementos gráficos no canvas.
O jogo permite que dois corremos no mesmo clima. O jogador 1 usa como teclas W e S, enquanto o jogador 2 usa como setas para cima e para baixo. Cada vez que a bola sai pela lateral de um jogador, o adversário ganha um ponto. O lugar é exposto no topo da tela.

Para testar, basta abrir no VS Code ou VS Codium e executar o arquivo index.html utilizando o plugin Live Server.
